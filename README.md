
# ToDo App

Todo Web Application for a client. Where the client is able to store its daily Task. 


## Features

- Add Task
- Update task
- Delete Task


## Run Backend

Clone the project

```bash
  git clone add git link
```

Go to the project directory

```bash
  cd server
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
